# Logo Setup

Place your logo files in this directory with the following names:

1. **dialog.png** - Dialog logo
2. **entc.png** - ENTC logo  
3. **partner.png** - Partner/Sponsor logo

## Recommended Specifications

- **Format**: PNG with transparent background
- **Dimensions**: Approximately 200x80 pixels (will be auto-scaled)
- **File size**: Keep under 100KB for optimal loading

## Notes

- The logos will be displayed in white tiles at the bottom of the left panel
- Each logo will be automatically scaled to fit within 35px height
- If a logo file is missing, that tile will be hidden automatically
